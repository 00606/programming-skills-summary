    • 编程技巧汇总

    为了方便快速编写代码，创建了这个公开库。旨在提高编写效率和编写质量，避免多次google一样的代码。这是一个在线公开笔记，也是一个提高自己能力的文档，会收集大量优秀的代码片段。

    Project maintained by xiaobeicn Hosted on GitHub Pages — Theme by mattgraham

    贡献方式：

    1. fork 项目，然后推送给我
    2. 创建 issues ,我会及时处理

    贡献要求

    1. 在对应文件夹内创建单独文件或在文件夹内README.md中直接扩充
    2. 文件内容不宜过大，最好控制在百行以内
    3. 如果是多个片段一个归属类，最好只创建一个文件
    4. 代码质量要高，可以是自己代码的提炼，也可以是开源优秀代码段
